Арматура для фундамента

Фундамент кладут между несколькими слоями земляного грунта, используя для этого либо стандартные буронабивные сваи, либо более сложные железобетонные плиты. Блочный фундамент имеет вид железобетонных блоков, которые вкопаны в слои грунтовых залежей. Выполняют такие конструкции из железобетона либо бетонных блоков.

После того, как бетонная плита полностью застынет, ее покрывают слоем штукатурки либо изразцов. После этого заливают асфальтом или песчано-гравийной смесью. Опалубку для ленточного фундамента делают в зависимости от того, для какой среды предназначено сооружение. Стоит отметить, что все размеры опалубки должны выдерживаться с запасом в 50 см.

Есть разные типы плит для ленточного фундамента. Это могут быть простые бетонные блоки, монолитные плиты и прочие. Как правило, боковины бетонных плит изготавливают с идентичной толщиной. Плиты для ленточного фундамента по высоте производятся в зависимости от размеров сооружения.

Нельзя допускать, чтобы данные плиты не были чуть шире, чем по 2 см. для того, чтобы плита свободно могла погружаться в почву. Армирование для ленточного фундамента выполняется из металлических прутьев, арматуры и блоков. Для того, чтобы лента начинала успешно использоваться, выполняется армирование специальным вязким раствором.

Для того, чтобы данный проект стал более полноценным, необходимо выполнить расчет. Он выполняется исходя из того, что строительство будет вестись под определенный тип почвы. Обратите внимание на расчет теплопотерь от земляных работ. Для решения этой задачи, необходимо рассматривать несколько вариантов расчета.
