Арматура для фундамента

Защитный слой бетона для арматуры в фундаменте

Защитный слой бетона для арматуры в фундаменте в инверторном исполнении состоял из перекрытия, арматуры с соединительной панелью и защитного слоя бетона в железобетонных частях каркаса. При этом на перекрытие укладывалась сетка-рабица из арматуры, также покрытая бетоном.

Фундамент из стеклопластиковой арматуры

Правила армирования Стеклопластиковая арматура, которая используется для укрепления фундамента, относится к числу самых современных материалов для армирования.

Арматура для ленточного фундамента

Ленточный фундамент может быть двух типов. В ленточном фундаменте верхние блоки производятся в заводских условиях, а в случае установки на забивных грунтах технология гораздо сложнее.
